# 📅 מערכת יומן חכם - Google Calendar PWA

מערכת ניהול יומן חכם עם תמיכה מלאה ב-CRUD ו-NLP בעברית לניהול אירועי Google Calendar.

## ✨ תכונות עיקריות

- 🎯 **8 פעולות CRUD מלאות**: יצירה, עדכון, מחיקה, העברה, שכפול, שינוי צבע, תזכורות, משתתפים
- 🇮🇱 **NLP בעברית**: הבנת פקודות טבעיות בשפה העברית
- 📅 **זיהוי תאריכים חכם**: "מחר", "יום רביעי", "15 בינואר", וכו'
- 🎨 **10 צבעים זמינים**: כחול, ירוק, אדום, כתום, סגול, צהוב, אפור, ורוד, טורקיז, ירוק בהיר
- ⏰ **תזכורות מרובות**: הגדרת תזכורות מרובות לכל אירוע
- 👥 **הוספת משתתפים**: הזמנת משתתפים באמצעות אימייל
- 🔍 **חיפוש חכם**: חיפוש אירועים לפי מזהה או כותרת
- 💬 **הודעות בעברית**: כל ההודעות והתגובות בעברית מפורטת
- 📱 **PWA**: אפליקציה ניתנת להתקנה על מכשירים ניידים

## 🚀 התחלה מהירה

### שלב 1: העלאת הקוד ל-Google Apps Script

1. פתח את [Google Apps Script](https://script.google.com)
2. צור פרויקט חדש
3. העתק את הקוד מ-`src/Code.gs` לקובץ Code.gs
4. העתק את התוכן מ-`src/appsscript.json` לקובץ appsscript.json
5. שמור את הפרויקט

### שלב 2: פרסום כ-Web App

1. לחץ על "Deploy" > "New deployment"
2. בחר "Web app"
3. הגדר:
   - **Execute as**: User accessing the web app
   - **Who has access**: Anyone
4. לחץ "Deploy"
5. העתק את כתובת ה-URL שקיבלת

### שלב 3: עדכון ה-Frontend

1. פתח את `index.html`
2. עדכן את כתובת ה-iframe לכתובת ה-URL שקיבלת:
```html
<iframe src="YOUR_SCRIPT_URL_HERE"></iframe>
```

### שלב 4: אישור הרשאות

1. בפעם הראשונה שתשתמש במערכת, תתבקש לאשר הרשאות
2. אשר גישה ליומן Google שלך
3. המערכת מוכנה לשימוש!

## 🔧 הגדרות

### הרשאות OAuth

המערכת דורשת את ההרשאות הבאות:
- `https://www.googleapis.com/auth/calendar` - גישה מלאה ליומן
- `https://www.googleapis.com/auth/script.external_request` - בקשות חיצוניות
- `https://www.googleapis.com/auth/userinfo.email` - זיהוי משתמש

### אזור זמן

המערכת מוגדרת לאזור הזמן `Asia/Jerusalem`. ניתן לשנות ב-`appsscript.json`.

## 📱 התקנה כ-PWA

האפליקציה תומכת ב-Progressive Web App:

### Android:
1. פתח את האתר בדפדפן Chrome
2. לחץ על "Install" או בחר "Add to Home Screen" מהתפריט

### iOS:
1. פתח את האתר ב-Safari
2. לחץ על כפתור השיתוף
3. בחר "Add to Home Screen"
