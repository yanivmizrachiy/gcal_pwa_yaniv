# gcal_pwa_yaniv

Smart Calendar Application - Hebrew PWA with Google Apps Script integration.

## Project Structure

- `/src` - Google Apps Script backend code
- `/frontend` - Next.js frontend application
- `/icons` - PWA icons
- `index.html` - Main PWA entry point
